# 目录结构

```

├── config/                      # 配置文件目录

├── docs/                        # VitePress 官方文档
│   ├── .vitepress/              # VitePress 配置
│   └── ...

│   ├── enshan/                  # 恩山论坛
│   ├── kanxue/                  # 看雪论坛
│   ├── sf/                      # 顺丰速运
│   ├── shyp/                    # 上海杨浦
│   ├── huaruntong/              # 华润通
│   ├── erke/                    # 鸿星尔克
│   ├── wps/                     # WPS Office
│   ├── smzdm/                   # 什么值得买
│   └── mipay/                   # 小米钱包
├── notification.py              # 统一推送模块
├── README.md                    # 项目说明
└── requirements.txt             # 依赖列表
```
