# 常见问题

## Cookie 失效怎么办？

Cookie 有有效期限制，失效后需要重新获取并更新配置文件。建议定期检查并更新。

## 签到失败如何排查？

1. 检查 Cookie 或 token 是否过期。
2. 查看日志中的错误信息。
3. 确认账号是否正常（未被封禁）。
4. 检查网络连通性。

## 如何添加新账号？

在 `config/token.json` 中对应平台的 `accounts` 数组中追加对象即可。

## 如何禁用某个平台？

- 删除 `config/token.json` 中对应平台配置。
- 或在青龙面板中禁用对应的定时任务。

## 推送没有收到？

检查以下内容：

1. 推送配置是否完整（`config/notification.json` 或环境变量）。
2. 推送服务是否可用、网络是否可达。
3. 脚本日志中推送模块的错误提示。

## 脚本执行报错怎么办？

1. 检查 Python 版本和依赖是否安装。
2. 确认配置文件 JSON 格式正确。
3. 确认脚本路径与定时任务路径一致。
